# jwttool

## Why?
I wanted to learn rust,<br>
I wanted to understand JWTs on a deeper level,<br>
and I wanted a good tool to tamper the JWTs with.<br>
So I started working on this tool

## Goal

The goal for this tool is to be able to do most JWT attacks that are useful when doing security testing. The tool should also be simple to use and provide other convenient features like decoding, fetching a public key from JWKs based on kid and iss.
